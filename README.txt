Easy Tracker - Navsingh Studio
Attendance System powered with QR code scanning system for replacing standard attendance systems in Colleges, Institutes and Companies.
Functionality:
1- One time login 
2- Open the app scan the qr code available in the class room/building
3- App saves the attendance data to the server 
4- The data can be queried by the admin of the system on back-end through web based dashboard.
